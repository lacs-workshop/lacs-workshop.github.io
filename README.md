
# LACS Workshop (International Workshop on Learning Augmented Compilers & Systems)

LACS (International Workshop on Learning Augmented Compilers & Systems) is a venue dedicated to advancing the integration of machine learning techniques into compilers and systems research.

The workshop brings together researchers and practitioners from both machine learning and systems communities to explore how learning-based approaches can improve the design, optimization, and performance of compilers, programming languages, and large-scale computing systems.

## Background

LACS continues the legacy of the previous workshop series:

**MLSH (International Workshop on Machine Learning for Software Hardware Co-Design)**
https://commit.csail.mit.edu/mlsh/

While MLSH focused broadly on machine learning for co-design across software and hardware stacks, LACS sharpens the focus toward compiler and systems challenges, emphasizing learning-augmented techniques.
