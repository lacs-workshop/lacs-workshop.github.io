LACS 2026

Learning-Augmented Compilers & Systems

Call for Papers

Overview

Recent advances in machine learning—particularly foundation models, graph neural networks, and reinforcement learning—are transforming how systems are designed, optimized, and deployed.

Modern systems are no longer purely engineered using heuristics; instead, they increasingly incorporate learned decision-making components, from compiler optimizations and kernel selection to scheduling and resource management.

LACS aims to bring together researchers and practitioners working at the intersection of:

  • Machine Learning
  • Compilers and Programming Systems
  • Computer Architecture and Hardware
  • Large-scale Infrastructure and Distributed Systems

The workshop focuses on learning-augmented system design, where ML models are integrated into the core of system decision-making pipelines.

Topics of Interest

Learning-Augmented Compilers

  • ML-guided compiler optimizations (LLVM, MLIR, Halide, TVM, XLA)
  • Learned cost models and scheduling (e.g., autoschedulers)
  • Kernel selection and operator optimization
  • ML-based profile-guided optimization (ML-PGO)
  • Transfer learning across architectures and workloads

LLMs for Systems

  • LLM-driven optimization, scheduling, and autotuning
  • Code generation for systems and kernels
  • Agent-based system optimization and debugging
  • ML-assisted performance modeling and analysis

Systems for ML / LLMs

  • Efficient training and inference systems
  • Serving systems (batching, caching, KV management)
  • Distributed systems for large-scale models
  • Memory and compute optimization

Hardware–ML–Compiler Co-Design

  • ML-driven hardware design and architecture search
  • Compiler–hardware co-optimization
  • Architecture-aware ML models
  • Cross-layer optimization

Submission Types

  • Extended Abstract (2 pages): Work-in-progress and early ideas
  • Position Papers (2–4 pages): Vision statements and future directions
